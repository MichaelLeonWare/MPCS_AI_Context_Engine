PROJECT NAME
────────────
MPCS — Michael's Portable Context System


PROJECT DESCRIPTION
───────────────────
MPCS is a lightweight portable context and memory system for moving
structured working context between LLM chat windows, models, and tools.

Drop a single Markdown file into any chat window — Claude, ChatGPT,
Gemini, or any model that accepts file uploads — and instantly restore
your working context, project rules, decisions, and handoff notes.

Built with a built-in security layer called the MPCS Librarian, which
scans every file on update and on load, classifying all content as:

  ✅ Inert data
  ⚠️ Possibly dangerous
  🚫 Malicious code

Nothing loads silently. Everything is flagged and summarized before it
enters your session. You are always in control.

Simple. Cheap. Controllable. Portable.

--- VERSION A: TECHNICAL AUDIENCE ---

I built a portable memory system for LLMs. It's called MPCS.

The problem: every time you open a new chat window, your context is gone.
Your preferences, your project rules, your decisions — all of it. You
start over. Every single time.

MPCS fixes that with a single Markdown file.

Drop mpcs_v1.md into any chat window — Claude, ChatGPT, Gemini, any
model that accepts file uploads. Your working context loads instantly.

But I didn't just want a context file. I wanted a safe one.

So I built the MPCS Librarian into the system. Every time you update
the file or load it into a new window, the Librarian scans and
classifies every piece of data:

  ✅ Inert data — loads fine
  ⚠️ Possibly dangerous — warns you before loading
  🚫 Malicious code — stops and alerts you

Nothing loads silently. Everything gets a flag and a summary. You stay
in control of what enters your session.

Six commands. One file. Works across models.

  /mpcs_load         → load your context
  /mpcs_update       → save your session
  /mpcs_export       → export for the next window
  /mpcs_librarian_scan → manual safety scan
  /mpcs_status       → check what's loaded
  /mpcs_clean_template → start fresh

Free for noncommercial use. Commercial licensing available.

--- VERSION C: PERSONAL / STORY-DRIVEN ---

I got tired of repeating myself to AI.

Every new chat window. Every new session. Every time I switched models.
Same context. Same background. Same preferences. Over and over.

So I built something.

MPCS — Michael's Portable Context System — is a single Markdown file
that carries your working context between any LLM chat window. Drop it
in. Type /mpcs_load. You're back where you left off.

I also built in a security layer I call the MPCS Librarian. Because
context files can carry risks — injected instructions, sensitive data,
or worse. The Librarian scans every file before it loads and flags
anything that isn't clean. You see the summary. You decide what loads.

It's lightweight. It's free for noncommercial use. And it works today
with Claude, ChatGPT, and Gemini.

Built under the Michael Leon Ware brand. First release: v1.
